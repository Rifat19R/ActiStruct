# Code of Conduct

## Our Pledge

Project participants are expected to help keep ActiStruct welcoming,
professional, and constructive for users and contributors from all backgrounds.

## Expected Behavior

- Use respectful and inclusive language.
- Focus technical discussion on reproducible evidence and actionable fixes.
- Be patient with users who are setting up scientific software for the first time.
- Credit prior work and external contributions clearly.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing private information without explicit permission.
- Repeated off-topic or disruptive comments after a maintainer asks for focus.

## Enforcement

Report concerns through the repository issue tracker or by contacting the
maintainer through the contact information in the repository metadata.
Maintainers may edit, hide, or remove comments and may restrict participation
when needed to protect the project community.

This code of conduct is adapted from common open-source community standards and
is intended to support a respectful scientific software project.
